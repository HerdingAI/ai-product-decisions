# Attribution

**Own work (D1).** All code in this artifact — the judge, bias probes, evaluation
pipeline, report, calibration set, and collection scripts — is original work. No forks,
no vendored third-party code, no external comparison arms, no outbound links to anyone
else's repository.

**Base app.** The responses under evaluation come from `agentic-copilot`, a companion
build (own work, MIT). This artifact evaluates that app; it does not modify it.

**Models.** All model calls go through **OpenRouter** (D6), the single model-provider
standard for this repo. The judge pass reported here used
`deepseek/deepseek-v4-flash` (`reasoning_effort=high`). No first-party inference keys
are used or committed; a reader reproduces the numbers with their own OpenRouter key.

**Data.**
- Calibration queries are self-authored (`eval_judge/calibration_set.py`).
- Responses are generated by `agentic-copilot` and collected verbatim
  (`labels/calibration_responses.json`).
- Human labels are **Carlos's own**, as sole labeler — disclosed openly, per the
  labeling rules in `RUBRIC.md` and sampling design in `SAMPLING.md`.

**No employer or business-outcome data (D15).** This artifact carries only reproducible,
demo-internal numbers (agreement rates, κ, cost, latency) that a stranger can regenerate
from the code. Nothing here reflects any employer's data, systems, or results.
